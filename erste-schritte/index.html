1. Ordnerstruktur

Hier ist ein Beispiel für eine Ordnerstruktur. Besonders wichtig ist die Bennenung der Homepage-Datei als index.html.

2. HTML Datei erstellen

Erstelle eine neue Datei mit dem Namen index.html in deinem Projektordner. Diese Datei wird die Startseite deiner Website sein.

3. Strukturierung

Nutze <h1>, <h2>, und andere Überschriften-Tags, um deine Inhalte zu strukturieren. Verwende Absätze <p> für Text und <a> für Links.

4. CSS Datei erstellen

Erstelle eine neue Datei mit dem Namen style.css in deinem Projektordner. Diese Datei wird das Aussehen deiner Website definieren.

Verlinke die CSS-Datei in deiner HTML-Datei im <head>-Bereich:

<link rel="stylesheet" href="style.css">

5. CSS Regeln hinzufügen

Füge in der style.css-Datei CSS-Regeln hinzu, um das Aussehen deiner HTML-Elemente zu gestalten. Zum Beispiel:

body {
background-color: black;
color: white;
}

Dies ändert die Hintergrundfarbe der Seite und die Textfarbe.

6. Vorschau im Browser

Öffne die index.html-Datei in einem Webbrowser, um eine Vorschau deiner Website zu sehen. Du kannst die Datei einfach per Drag & Drop in das Browserfenster ziehen.

Alternativ kannst du mit Visual Studio Code und der Erweiterung "Live Preview" eine Vorschau starten.

7. CSS-Box-Modell

Das CSS-Box-Modell beschreibt, wie Elemente auf einer Webseite dargestellt werden. Jedes Element wird als Rechteck (Box) dargestellt, das aus vier Bereichen besteht:

  • Content: Der eigentliche Inhalt des Elements (Text, Bild, etc.).
  • Padding: Der Innenabstand zwischen dem Inhalt und dem Rand des Elements.
  • Border: Der Rahmen um das Element.
  • Margin: Der Außenabstand zwischen dem Element und anderen Elementen.
