Erste Schritte

1. Ordnerstruktur

Hier ist ein Beispiel für eine Ordnerstruktur.

  • Startseite: index.html
  • Stylesheet: style.css
  • Bilder: images/

2. HTML-Datei erstellen

Erstelle eine neue Datei mit dem Namen index.html in deinem Projektordner. Diese Datei wird die Startseite deiner Website sein.

In Visual Studio Code kannst du dieses Grundgerüst einfügen, indem du ! eintippst und Enter drückst.

3. CSS-Datei erstellen und linken

Erstelle eine neue Datei mit dem Namen style.css in deinem Projektordner. Diese Datei wird das Aussehen deiner Website definieren.

Verlinke die CSS-Datei in deiner HTML-Datei im <head>-Bereich:

<link rel="stylesheet" href="style.css">

4. Strukturierung

Teile deine Website in <header>, <main> und <footer> auf, um eine klare Struktur zu schaffen. Verwende <h1>, <h2>, und andere Überschriften-Tags, um deine Inhalte zu strukturieren. Verwende Absätze <p> für Text und <a> für Links.

5. Dividers und Classes

Verwende <div>-Elemente, um verschiedene Bereiche deiner Seite zu unterteilen. Du kannst auch Classes verwenden, um Stile auf mehrere Elemente anzuwenden.

Um eine Class im CSS zu definieren, verwende den Punkt . gefolgt vom Namen der Class: .example {}

6. CSS-Box-Modell

Das CSS-Box-Modell beschreibt, wie Elemente auf einer Webseite dargestellt werden. Jedes Element wird als Rechteck (Box) dargestellt, das aus vier Bereichen besteht:

  1. Content: Der eigentliche Inhalt des Elements (Text, Bild, etc.).
  2. Padding: Der Innenabstand zwischen dem Inhalt und dem Rand des Elements.
  3. Border: Der Rahmen um das Element.
  4. Margin: Der Außenabstand zwischen dem Element und anderen Elementen.

7. Vorschau der Website

Um deine Website in einem Webbrowser anzusehen, öffne die index.html-Datei in einem Browser.

In Visual Studio Code kannst du auch die "Live Server"-Erweiterung verwenden, um Änderungen an deiner Seite in Echtzeit zu sehen.

8. Nächste Schritte

Jetzt kannst du beginnen deine Website mit weiteren HTML-Tags und CSS-Attributen zu gestalten.
